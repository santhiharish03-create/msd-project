# Sponsors

Your support helps us maintain a reliable, student- and faculty-friendly timetable experience.

## Why sponsor
- Faster iteration on features like conflict detection and real-time updates
- Dedicated time for accessibility, performance, and documentation
- Long-term maintenance and security updates

## How to sponsor
- GitHub Sponsors (if enabled on this repo)
- Institutional support (departments or offices)
- One-time support or services funding

If GitHub Sponsors isn’t enabled yet, please reach out and we’ll coordinate:
- Email: timetable@vignan.ac.in
- Or open a Discussion/Issue to start the conversation

## Recognition
Sponsors can be optionally listed here (individuals or institutions). If you’d like recognition, include the preferred name and link.
